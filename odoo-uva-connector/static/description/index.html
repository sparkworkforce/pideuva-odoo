Uva PR Connector

Integrate Odoo POS and delivery with Uva PR — Puerto Rico's local delivery platform

Flow A — Incoming Orders (POS Connector)

Uva customer orders appear automatically in the Odoo POS interface. Staff can accept, reject, or mark items as unavailable directly from POS — without touching the Uva app.

  • Webhook + polling fallback ingestion
  • Real-time POS notification via bus.bus
  • Configurable auto-accept timeout
  • Multi-store support
  • Product mapping table

Flow B — Outbound Delivery (Uva Fleet)

Dispatch deliveries via Uva Fleet drivers directly from Odoo sale orders or stock pickings. Real-time tracking status posted to Odoo chatters.

  • Cost estimate before dispatch
  • Dispatch from sale order or stock picking
  • Real-time status tracking
  • Cancellation from Odoo
  • Automatic retry on transient failures

Compatibility

Supports Odoo 17, 18, and 19 Enterprise. Deployable on Odoo.sh.

Demo Mode

Enabled by default. Evaluate the full integration without real Uva credentials. Disable demo mode and enter your API keys when ready for production.
